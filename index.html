Autoprefixer

Autoprefixer

  • возьмет данные с caniuse.com
  • расставит нужные префиксы
  • уберет ненужные
  • gradient, flexbox, :fullscreen, @keyframes, background-size, transform

Проблемы CSS

  • вертикальное центрирование
  • колонки одинаковой высоты
  • верстка для IE6
  • хаки

Настоящие проблемы CSS

  • непредсказуемый каскад
  • вес селекторов
  • удаление старого кода

Нет уверенности, что изменения в стилях не сломают что-то еще.

Люди боятся вносить изменения, потому что не могут предсказать последствия.

				.attachments {
				  margin: 1em 0;
				}

				// Override margin of attachment class,
				// so it doesn't mess up page layout.
				body.attachments {
				  margin-top: 0;
				}
			
				@media (min-width: 1440px) {
				  .projects {
				    ul.tests {
				      li.test .contents {
				        .inner_contents {
				          .description_container {
				            width: 50%!important;
				          }
				        }
				      }
				    }
				  }
				}
